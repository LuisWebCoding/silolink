# 🌾 Silo Link - Frontend

Aplicação web desenvolvida em **React** para o gerenciamento e monitoramento de silos de sementes. O sistema permite o cadastro, edição, visualização e exclusão de silos, com foco em uma interface limpa e intuitiva.

## 🚀 Tecnologias Utilizadas

* **React** (Biblioteca de interface)
* **Vite** (Build tool rápida)
* **React Router DOM** (Navegação/Rotas)
* **CSS Puro** (Estilização global e componentes)

---

## 📦 Como Abrir e Rodar o Projeto

Siga estes passos para executar a aplicação no seu computador:

### 1. Pré-requisitos

Certifique-se de ter o **Node.js** instalado na sua máquina.
* Para verificar se você tem, abra o terminal e digite: `node -v`

### 2. Instalação das Dependências

Se você baixou o projeto ou clonou do Git, a pasta `node_modules` (que contém as bibliotecas) geralmente não vem junto. Você precisa restaurá-la.

1.  Abra o terminal (CMD, PowerShell ou Terminal do VS Code).
2.  Navegue até a pasta raiz do projeto:
    ```bash
    cd silo-link
    ```
3.  Execute o comando para baixar as bibliotecas:
    ```bash
    npm install
    ```

### 3. Executando o Projeto

Após instalar as dependências, inicie o servidor de desenvolvimento:

```bash
npm run dev
